===========================================================
  أهلاً بك في دليل تثبيت وتشغيل المتجر الإلكتروني والمساعد الصوتي 🚀
===========================================================

هذا المجلد يحتوي على كافة الملفات المحسنة وجاهزة للنشر المباشر على خوادم Ionos أو أي بيئة استضافة أخرى:

📁 محتويات المجلد:
----------------
1. [server.cjs] - ملف المساعد الصوتي والخادم المحوسب مدمجاً ومحسناً بشكل كامل.
2. [package.json] - يحتوي على التعريفات البرمجية والتشغيلية وأهم التبعيات اللازمة لتشغيل الخادم.
3. [.env.example] - مثال لتهيئة البيئة (انسخه إلى .env وضع فيه مفتاح GEMINI_API_KEY الخاص بك).
4. [index.html] و [assets/] - ملفات الموقع الثابتة والوجهية وتصميم الويب المتقدم.
5. [images/] - كافة صور المنتجات المحلية وصورة المؤسس (Ayman Ali Hammad).

⚡️ طريقة التشغيل السريعة على Ionos (باستخدام Node.js App or SSH):
--------------------------------------------------------------
1. قم بفك الضغط عن الملفات داخل مجلد تطبيق الـ Node.js المخصص في لوحة Ionos.
2. انسخ الملف [.env.example] إلى ملف جديد باسم [.env] وقم بكتابة مفتاح الـ API الخاص بـ Gemini:
   GEMINI_API_KEY=مفتاح_الجيميني_الخاص_بك_هنا

3. افتح الطرفية (SSH Terminal) واكتب الأوامر التالية لتثبيت الملحقات وتشغيل التطبيق بشكل دائم:
   
   # أ) تثبيت التبعيات الضرورية للإنتاج:
   npm install --omit=dev

   # ب) تشغيل الخادم بالمنفذ 3000 ببرنامج إدارة العمليات ليبقى نشطاً دائماً:
   pm2 start server.cjs --name "voice-assistant-store"

🔒 تلميح تقني مهم بخصوص نظام الـ WebSockets:
------------------------------------------
لنجاح عمل المساعد الصوتي، تأكد من أن خادم الويب (Nginx, Apache, or Routing System) في Ionos يمرر حركة مرور الـ WebSockets (ترقية الاتصال Upgrade / Connection) للمسار [/live].
